Telegram vs Telegram X：该选择什么

2023-07-02 分类：电报使用 评论(0)

尽管 Telegram 是技术最先进的通讯工具之一，但帕维尔·杜罗夫 (Pavel Durov) 领导的开发团队并不止于此。2018年1月，官方向用户展示了Telegram X客户端，其功能与原来的Messenger相当，并为工作提供了额外的便利。

Telegram X 的好处

Telegram X 是在全新的代码库上从头开始编写的。这使得他的工作速度比原来的信使还要快。

尤其是运行 iOS 操作系统的手机用户会欣赏这一优势。此外，在性能较低的 Android 设备和 64 位小工具型号上，速度的改进将非常明显。

此外，还改进了界面、设计，并添加了原始版本中没有的新功能，并方便访问活跃的聊天室和频道。

注意力！纠正 IOS 的代码错误是当务之急，因为 TDLib 库与苹果技术的兼容性导致了问题。因此，信使速度的差异是苹果技术拥有者首先注意到的。

此外，这些应用程序之间的区别在于 Telegram X 可以立即从 AppStore 或 Google Play 下载。但是，网络客户端仍然免费，就像原来的即时通讯工具一样。

与 Telegram 的主要区别

考虑一下 Telegram 和 Telegram X：它们之间有什么区别，以及用于比较的基本参数。

编程语言

Telegram X 应用程序使用 Swift 编程语言而不是传统的 C 语言编写。

该产品的创新之处在于它与Cocoa和Cocoa Touch框架的兼容性，以及苹果标准Objective-C和Objective-C的使用。因此，这提供了更少量的代码，从而显着提高了所有类型设备上的下载速度。您不仅可以在应用程序打开期间注意到差异，而且还可以在平滑度、动画速度方面注意到差异。

能源效率

Telegram X 在后台消耗的电量比 Telegram X 本身要少得多。如果官方网络客户端即使不活跃也需要 1% 的费用，那么原来的 Messenger 则需要 14%。

主题

首先，当熟悉 Telegram X 时，值得注意的是，它不再是带有信使名称的徽标，而是添加了一个能够在消息和通话之间切换的选项。此外，固定聊天和其他联系人之间出现了分隔符。

此外，邀请用户选择四个主题之一：

  • Day Classic – 标准 Telegram 主题，默认启用；
  • Day – iMessage 苹果应用程序的传统设计；
  • 夜晚——黑白主题，只有图像保留彩色；
  • 夜蓝色——黑色背景上的深蓝色界面。

此外，在调色板上，还可以设置整个设计主题的整体色调。界面设计将保持不变，只有背景颜色与消息框和复选标记的兼容性会发生变化。

重要的！在设置中，您可以激活自动包含夜间模式，这将与现实中的黄昏时间一致。

夜间主题所需的电力比白天少得多。这在不能很好地充电的低性能设备以及高科技设备上非常明显。

这一创新在配备 OLED 显示屏的 iPhone X 上尤为明显。该屏幕的一个特点是每个嵌入像素的自主操作，发出一定量的光以创建公共图像。

对于黑色的外观，所需的像素数量会简单地褪色：这对人眼来说很方便，此外，还可以显着节省电池电量。

设置

Telegram X 与原来的即时通讯软件一样，配备了两个设置菜单窗口：第一个在向左滑动时打开，第二个在选择“设置”选项时出现。对比一下，你立刻就能发现两个窗口都发生了明显的变化。

在左侧菜单中，头像被放置在标题中，图像也分别放大，并且帐户中可用的功能数量减少。同时，在底行添加了两个开关：用于夜间主题和包含“面团”。

参考！“假人”被称为消息框，放置在半圆形框架中。Telegram 中的聊天和频道中的所有消息均采用此设计，Telegram X 中也默认启用此设计。但是，官方 Web 客户端提供了自己的界面版本，其中消息以无框形式显示为单个文本和媒体图像流。

主菜单与原来的菜单不同，增加了一些新功能：

  • 改变字体大小；
  • 纳入“面团”；
  • 格式化文本 Markdown（粗体、斜体）；
  • 默认将视频消息摄像头从前置切换到主摄像头；
  • 数据节省程序（切换到 3G、4G、LTE 或漫游时节省移动流量：媒体文件自动下载停止）。

Telegram X 缺少什么

Telegram 和 Telegram X 网页客户端的区别不仅在于增加了新选项，还在于缺少官方即时通讯工具具有的一些功能：

  • 跨平台（电脑上不支持）；
  • 同步；
  • 安全的点对点通话；
  • 链接不是在应用程序内部打开，而是在默认启用的浏览器中打开；
  • 付款信息未删除等

Telegram 与 Telegram X：哪个更好？

Telegram 与 Telegram X 的主要区别在于代码的技术结构。当 Telegram X 首次出现在数字市场时，它的优势引起了人们的极大兴趣。几年后，网络客户端和即时通讯工具之间的竞争实际上已经消失，许多功能已经变得相同。

在 2019 年底，使用某个特定版本并不重要，除非是性能不够高的智能手机对 Telegram 常规版本的支持不佳的情况。

因此，这两款应用程序在 Android 和 iPhone 上的运行都具有令人羡慕的稳定性。绝大多数移动设备都支持它们。

如果您只需要选择 Telegram 或只选择 Telegram X，那么您应该停止使用标准版本，因为数据安全高于一切。如果节省电量、释放内存和流量是更重要的任务，那么 Telegram X 更合适。
