如何在 Twitter 上使用代码生成器 2FA（分步说明）

2025年 3月 5日 作者 推特购买平台

本文介绍了 Twitter（现为 X）的双因素身份验证 (2FA) 代码生成器应用程序、如何启用 2FA 及其重要性。

2FA 为您的 X 帐户增加了额外的安全层。当您在 Twitter（现为 X）帐户上启用 2FA 时，登录将需要额外的身份验证，而仅凭密码无法完成此操作。

Twitter 允许使用短信（仅限高级订阅者）、身份验证器应用程序或 Yubikey 等物理设备进行 2FA。  

实现这种额外安全性的最方便且最具成本效益的方法是使用 Twitter 的代码生成器应用程序。因此，让我告诉你如何在 Twitter 上启用双因素身份验证以及一些可帮助你实现此目的的应用程序。

关于 Twitter 2FA

双重身份验证 (2FA) 确保只有帐户的合法所有者才能访问它。

它要求额外的 Twitter 身份验证步骤，这意味着需要输入两条信息才能登录：

  • 密码 
  • TOTP（基于时间的一次性密码）

这样一来，恶意行为者就不太可能未经授权访问您的 X 帐户。

注意：即使通过短信进行双重身份验证，也比没有双重身份验证要好。

使用 X 代码生成器应用程序的好处（以前称为 Twitter）

有以下几个理由要求我们在任何地方都部署双因素身份验证，Twitter 也不例外：

  • 只有高级 Twitter 用户才允许使用基于 SMS 的 2FA，并且拥有像 YubiKey 这样的物理设备也需要额外的费用，而代码生成器应用程序是免费的。
  • 2FA 可以在设备被盗等紧急情况下发挥作用，防止未经授权的访问。这对于严重依赖社交媒体账户保护品牌形象的公司或影响者来说至关重要。
  • 双因素身份验证系统是开始超越密码的好方法。如果有人仍在使用弱密码，那么双因素身份验证系统就成为不可避免的必需品。

如何在 X (Twitter) 上启用 2FA

使用 X 的双重身份验证非常简单。您可以在 Mac、Windows、Android 或 iOS 上进行此操作。以下是在任何 Web 浏览器上进行设置的分步指南：

  • 登录您的 X 个人资料，然后单击左侧边栏中的“更多”。
  • 选择“设置和隐私”。
  • 点击“安全和账户访问”后，会出现更多选项。在展开的菜单中点击“安全”。
  • 点击“双重身份验证”。
  • 从 2FA 验证选项中选择“身份验证应用程序”。
  • 如果您选择“身份验证应用程序”，Twitter 将要求您输入密码。输入密码后，请在以下屏幕上单击“确认”和“开始”。
  • 随后，Twitter 会向您提供一个 二维码 ，您可以使用任何 2FA 应用程序进行扫描。
  • 最后，通过输入最近生成的 OTP，将您的 Twitter 帐户链接到您选择的身份验证器应用程序。为了安全起见，请单击获取备份代码并将其保存在安全的地方，以便在没有 2FA 设备的情况下登录。

适用于 X（以前称为 Twitter）的最佳 2FA 代码生成器应用程序

#1. 2FA 身份验证器 (2FAS)

2FAS 是一款出色的 Twitter 开源代码生成器应用，显然也适用于其他应用。您可以扫描二维码或手动输入您的令牌信息。

完成后，您每次输入密码时都会检索一个令牌。

这款应用的用户界面简洁干净。它支持大多数需要双因素身份验证的网站的 TOTP 和 HOTP 协议。这款应用跨平台，有 iOS 和 Android 应用以及浏览器扩展程序。

下载：  Android  |  iOS

#2. Google 身份验证器

Google Authenticator 是 Google 推出的一款 2FA 应用程序，可以帮助您保护在线帐户（包括 Twitter）的安全。

Google Authenticator 是一款非常用户友好的应用程序，几乎每个接受基于 TOTP 的应用程序的网站都会支持它。 

Google 确认不会与第三方共享数据，并且始终使用安全连接。总体而言，它易于使用、可靠，并且支持几乎所有门户网站。

下载：  Android  |  iOS

#3. Microsoft Authenticator

Microsoft Authenticator 为 Twitter 提供安全可靠的 2FA 代码生成。

这是一款简单的应用程序，它为每项接受 TOTP 的服务生成唯一代码，无需网络连接。然而，这背后还有更多秘密。

您可以使用该应用程序无需密码即可登录 Microsoft 应用程序，包括 OneDrive 和 Office 365。根据设备支持，该应用程序允许用户使用指纹、面部或任何其他形式的身份验证来验证其身份。Microsoft Authenticator 还通过 Android 和 iOS 上的 Microsoft 服务器提供云备份。

Microsoft 帐户支持意味着您无需输入代码来批准登录，并且可以通过推送通知无缝登录。

下载：  Android  |  iOS

#4. Twilio Authy 身份验证器

Twilio Authy应用  是最受欢迎的双因素身份验证应用之一。它是跨平台的，具有适用于 Android、iOS、Linux、macOS 和 Windows 的原生应用。

它还支持一系列协议（TOTP、推送通知等）和网站，并且可以轻松备份。

使用 Authy，您可以轻松保护您的在线帐户。您可以像使用任何其他应用程序一样携带令牌。它还会加密备份。

Authy 使用的算法与 NSA 保护其超级机密所用的算法相同。

Twilio Authy 是一款功能强大的实用程序应用程序，它拥有许多功能，并因其跨平台可用性而引人注目。

下载：  Android  |  iOS

从 iOS 15 开始，Apple 在 iCloud Keychain 应用中添加了原生 2FA 支持。现在，您也可以使用它来输入基于时间的代码来登录 Twitter。

使用 iCloud Keychain 代码生成可以说更安全，因为这家库比蒂诺科技巨头以其用户隐私和安全而闻名。此外，它会同步到您的 iCloud 帐户，提供无缝可用性。例如，我可以使用安装在 Windows 计算机上的 iCloud 应用程序上生成的 TOTP 代码。

最后的话

设置额外的 Twitter（现在为 X）身份验证将提高您帐户的安全性。但是，即使启用了双因素身份验证，最佳做法也是使用强密码。定期更改密码并使用密码管理器是最佳选择。

此外，您还可以将 2FA 与其他安全措施（例如生物识别认证）相结合，使 2FA 更加有效。
